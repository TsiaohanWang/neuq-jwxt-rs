1//! > 本项目仍在开发中，可能会出现多次破坏性更新！请谨慎留意你所使用的版本！
2//! 
3//! `neuq-jwxt-rs` 是一个提供能与[东北大学秦皇岛分校教务系统](https://jwxt.neuq.edu.cn/eams)交互的 HTTP 客户端和数据处理的库。
4//! 
5//! ## 功能
6//! 
7//! 目前提供：
8//! 
9//! - 本地 `.env` 文件的读取；
10//! - 教务系统的个人登录；
11//! - 对教务系统“空闲教室查询”功能的：
12//!   - 数据解析与过滤，以及
13//!   - 支持文件读写的数据结构。
14//! 
15//! 未来将会实现：
16//! 
17//! - [ ] 对 Cloudflare Workers 运行环境的支持；
18//! - [ ] 对教务系统“我的课表”功能的：
19//!   - 数据解析与过滤，以及
20//!   - 支持文件读写的数据结构。
21//! 
22//! ## 快速开始
23//! 
24//! 当你在本地运行时，请创建 `.env` 文件并执行：
25//! 
26//! ```bash
27//! echo NEUQ_USERNAME=[你的教务系统学号] >> .env
28//! echo NEUQ_PASSWORD=[你的教务系统密码] >> .env
29//! ```
30//! 
31//! 然后在异步 `main` 函数中使用 `env_var` 即可进行读取。
32//! 
33//! 接着使用 `NeuqClient` 提供的 `new` 和 `login` 方法即可完成登录。
34//! 
35//! ### 示例
36//! 
37//! 以下示例演示了如何读取本地 `.env` 文件中的环境变量，并使用新建 HTTP 客户端登录教务系统。
38//! 
39//! ```rust
40//! let (usn, pwd) = login_var()?;
41//! 
42//! let client = NeuqClient::new()?;
43//! let res = client.login(usn.as_str(), pwd.as_str()).await?;
44//! 
45//! match res {
46//!     true => info!("Login successful."),
47//!     false => {
48//!         error!("Login failed.");
49//!         return Err(anyhow::anyhow!("Login failed."));
50//!     }
51//! }
52//! ```
53//! 
54//! 可在 `./src/examples` 中找到更多应用示例。
55//! 
56//! ## 版本支持
57//! 
58//! 本项目最低支持的 Rust 版本（MSRV）为 `1.93.1`。
